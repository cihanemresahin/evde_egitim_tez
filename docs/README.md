# 📄 Tez Dokümanları

> Proje planları, mimari diyagramlar, akış şemaları ve test raporları.

## 📁 Alt Dizinler

| Dizin | Açıklama |
|-------|----------|
| `flow_diagrams/` | Mermaid akış şemaları (sistem, watchdog, güvenlik) |
| `architecture_diagrams/` | Mimari diyagramlar ve görseller |
| `test_reports/` | M9 güvenlik testi ve entegrasyon test sonuçları |

## 📝 Kullanım

- Mermaid dosyaları `.md` formatında kaydedilir
- [Mermaid Live Editor](https://mermaid.live/) ile PNG/SVG'ye dönüştürülebilir
- Tüm diyagramlar tez raporunda kullanılmak üzere hazırlanmıştır
